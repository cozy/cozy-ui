A simple way to show how many *things* there are.

### Default

```
import Counter from 'cozy-ui/transpiled/react/Counter';
<div >
  <Counter count={42} />
</div>
```

### Maxed out

Above a certain number, an approximation is displayed. The default treshold is 99.

```
import Counter from 'cozy-ui/transpiled/react/Counter';
<div >
  <Counter count={14} max={5} />
</div>
```
